# 🎓 Student Management System (SQL Project)


# 📌 Overview

This project is a Student Management System built using SQL. It demonstrates how to design and manage a relational database with multiple tables and perform data analysis using queries.

# 🛠️ Technologies Used
SQL (MySQL)
Database Design

# 📊 Database Structure
The project consists of the following tables:
Students
Courses
Enrollments
Marks

# 🔗 Key Concepts Used
Primary Key (PK)
Foreign Key (FK)
Joins (INNER JOIN)
Group By & Aggregation
Many-to-Many Relationship

# 📂 Features
Manage student and course data
Track student enrollments
Store and analyze marks
Perform queries to get insights like:
Average marks
Top-performing students
Course-wise performance

# 📈 Sample Queries
Retrieve student performance using JOINs
Calculate average marks using GROUP BY
Find top students based on marks

# 🧠 Learning Outcome
Through this project, I understood how real-world databases are structured and how SQL queries are used for data analysis.
